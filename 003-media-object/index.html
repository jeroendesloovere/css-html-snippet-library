003 Media object — object oriented classes for image and text combinations.

Combinations:

  1. .media = generic media class (set clearfix and spacing)
  2. .media--block = image and text as blocks on top of each other
  3. .media--list = image and text next to each other
  4. .media__img = image container (left aligned)
  5. .media__img--rev = reversed image container (right aligned)
  6. .media__body = text container
